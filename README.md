# CATV 전송망 통합 조회 웹사이트

현장 스마트폰에서 CELL(셀명)과 B2C(전용선명)를 조회하고 국사, 선번정보, 평면도, 직선도를 확인하는 로컬 웹사이트입니다.

## 실행 방법

1. `웹사이트 실행.cmd`를 더블클릭합니다.
2. 서버가 백그라운드에서 실행되고 웹사이트가 자동으로 열립니다.
3. 직접 접속할 때는 `http://127.0.0.1:8000/`을 사용합니다.

터미널에서 실행할 때는 이 폴더에서 `npm start`를 입력해도 됩니다.

`index.html`을 직접 더블클릭해도 선명한 벡터 직선도를 사용하는 서버 화면으로 자동 연결됩니다. 서버가 꺼져 있으면 `웹사이트 실행.cmd`를 먼저 실행하세요.

## 기본 계정## 주요 기능

- CELL과 B2C 분리 조회
- 선번정보와 국사 평면도 조회
- 엑셀 원본 기반 벡터 직선도 표시
- 6글자 연속 일치 검색 위치 강조
- 큰 직선도 자동 확대 및 검색 위치 중앙 이동
- 스마트폰 두 손가락 확대·축소와 한 손가락 이동
- 가입 DB, 데이터 DB, 국사 평면도 DB, B2C 전용선 DB 관리
- 관리자 DB별 전체 삭제

## 주요 파일

- `index.html`: 웹사이트 화면
- `styles.css`: PC·스마트폰 화면 디자인
- `app.js`: 조회, DB, 평면도와 직선도 기능
- `server.mjs`: 로컬 웹서버
- `assets/`: 안성국사 직선도 및 벡터 렌더링 자료

## GitHub Pages 배포

저장소의 `main` 브랜치에 파일을 올리면 `.github/workflows/pages.yml`이 웹사이트 파일만 묶어 GitHub Pages에 배포합니다.

배포 전 주의사항:

- GitHub Pages 주소는 인터넷에 공개될 수 있습니다.
- 직선도와 검색 인덱스에 현장 정보가 포함되어 있으므로 저장소 공개 범위를 확인하세요.
- 런타임 로그와 임시 작업 폴더는 `.gitignore`로 업로드에서 제외됩니다.
