데이터 DB

국사 평면도 DB

노드명 또는 랙 정보를 입력한 뒤 이미지에서 해당 OFD 함·랙의 중앙을 클릭하세요.

목록의 수정 버튼으로 기존 이미지와 좌표를 불러올 수 있습니다. 좌표명을 선택한 뒤 이미지의 새 위치를 클릭하면 이동하며, 이미지가 잘못된 경우에만 새 파일을 선택하세요.

선번장(직선도) DB

선번장의 G열 노드명과 같은 직선도 시트를 그림으로 저장합니다. 기본 국사는 남색 도형 안의 셀명/B2C를 검색하고, 마평국사는 도형 밖의 검은색 셀명과 보라색 B2C(전용선명)를 검색 DB로 사용합니다. 검색어는 공백·기호를 제외하고 6글자 연속 일치로 찾습니다.

공용 배포 DB
